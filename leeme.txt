1- abre toda la carpeta desde vsc y asegurate de tener instalado python superior a 3.13 en el pc.
2- crea el entorno virtual venv desde powershell con "python -m venv venv"
3- activa el entorno virtual desde bash con "source venv/Scripts/activate"
4- instala flask y python con "pip install flask" "pip install python"
5- ejecuta la api con "python app.py"
6- copiar el bot ex5 en carpeta de expertos en la instancia de mt5
7- agregar en las preferencias de mt5 la siguiente url "http://127.0.0.1:5000/guardar_datos"
